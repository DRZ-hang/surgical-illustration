# 解剖学标准配色规范 (Standard Anatomy Colors)

> [!IMPORTANT]
> 遵循公认的解剖学配色传统可大幅降低读者的认知负担。

| 结构名称 (Structure) | 推荐 Hex 色号 | 视觉含义 (Significance) |
|-------------------|--------------|------------------------|
| **动脉 (Artery)** | `#E53935` | 鲜红，富含氧气的血液 |
| **静脉 (Vein)** | `#1A237E` | 暗蓝，含氧量较低的血液 |
| **神经 (Nerve)** | `#FDD835` | 亮黄，信号传导的象征 |
| **淋巴管 (Lymphatic)**| `#81C784` | 浅绿，与循环系统区分 |
| **骨骼 (Bone)** | `#FFF9C4` | 象牙/奶黄，坚硬质感 |
| **肌肉 (Muscle)** | `#D32F2F` | 深红/铁锈红，丰富的肌红蛋白 |
| **脂肪 (Fat/Lipid)** | `#FFECB3` | 浅黄，能量储备 |
| **结缔组织 (Tendon)**| `#ECEFF1` | 浅灰白，高致密胶原 |
| **皮肤 (Skin)** | `#FFCCBC` | 肉粉，表皮特征 |
| **胆管 (Bile)** | `#33691E` | 深绿，肝胆系统标识 |
| **尿液/输尿管** | `#FFEE58` | 透明黄 |

---

## 🎨 细节建议

1. **多重血管区分**:
   - 当图中有多个分支时，靠近心脏的用深色，分支用稍浅的颜色。
2. **断面处理**:
   - 肌肉断面应显示纤维纹理（Striae）。
   - 骨骼断面应区分皮质骨（更亮）和松质骨（有蜂窝状细节）。
3. **肿瘤组织**:
   - 在解剖背景中，通常给肿瘤分配一个与周围组织不协调的颜色（如深紫 `#4A148C`），并增加表面不规则性（Irregular texture）。
